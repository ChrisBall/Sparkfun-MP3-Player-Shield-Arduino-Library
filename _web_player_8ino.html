Detailed Description

Example sketch of using the MP3Shield Arduino driver to create a webserver.

Remarks
comments are implemented with Doxygen Markdown format
Author
Michael P. Flaga
AdaFruit

This sketch is a quick merge of http://github.com/adafruit/SDWebBrowse with SFEMP3Shield library demonstrating the VS1053 commanded through a web browser.

Warning
Need to set the following in the SFEMP3ShieldConfig.h, in order to work.
#define USE_MP3_REFILL_MEANS USE_MP3_Polled
as the Ethernet library makes interrupts not possible. any one with a fix, let me know.
